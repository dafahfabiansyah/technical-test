
# Project Title

technical test questions from PT MG Indotech
## Installation
Install with npm

```bash
git clone [https://github.com/dafahfabiansyah/technical-test.git](https://github.com/dafahfabiansyah/technical-test.git)
```

backend (express js)
```bash
cd backend
    npm install 
    nodemon index.js
    .env
    MONGO_URI=mongodb://127.0.0.1:27017/infotech
    PORT=5000
```

frontend (react)
```bash
cd frontend
    npm install 
    npm run dev
```
